Architecture
Clean Architecture
Domain → Application → Infrastructure → Web
Database
PostgreSQL via Npgsql + EF Core, soft delete with AuditableEntity
Authentication
JWT Bearer with cookie, global AuthorizeFilter, 401 → /Account/Login
JSON Serialization
System.Text.Json with camelCase. Use @Json.Serialize() or controller Json()
UI Pattern
Bootstrap 5, card shadow-sm, form-control-sm, operator-page layout
Voyage Formulas
SeaDay=Dist/Speed/24, ETB=ETA+2h, ETD=ETB+PortStay, NextETA=prevETD+prevSeaDay
Schedule Viewer
Flattens Voyage+VoyagePorts into POL→POD leg rows, needs 2+ ports
Learnings & Gotchas
Npgsql Timestamps
Npgsql 6+ rejects DateTime with Kind=Unspecified. Fix: EnableLegacyTimestampBehavior
JSON Casing
Never use System.Text.Json.JsonSerializer.Serialize() for Razor — it uses PascalCase
Date Inputs
Use dateStr.substring(0,16) to strip timezone. new Date() shifts values
ExceptionHandlingFilter
Returns HTML for ALL exceptions. API endpoints need try-catch returning JSON
Auth & Fetch
401 redirects to login (302). Check resp.ok AND content-type before .json()
Build Lock Errors
MSB3021/MSB3027 when app running — not code errors. .cshtml changes need no restart
Selector Consistency
When changing HTML structure, update ALL querySelectorAll selectors
DB Column Changes
EF Core won't auto-migrate. Need manual SQL: ALTER TABLE DROP COLUMN
URL Generation
Use @Url.Action() for attribute-routed controllers. Avoid hardcoded URLs
